# Phase 0.12 — QA, Beta Readiness, And Exit-Metric Review

## Scope

Phase 0.12 creates the checkpoint for deciding whether the Phase 0 wedge is ready for real beta users.

## Shipped

- Shared Phase 0 exit targets from the PRD.
- `/qa` founder review page with:
  - readiness percentage
  - founder test gates
  - Phase 0 PRD exit targets
  - current local MVP signals
  - known gaps before real beta
- Page-view analytics event for the QA screen.

## Founder Test Path

1. Open `/qa`.
2. Review the readiness percentage and test gates.
3. Create saves, booking intents, and reward unlocks in the app.
4. Return to `/qa` and confirm session proof numbers change.
5. Use the known gaps list as the pre-beta operating checklist.

## Exit Criteria Reference

- 500 beta users.
- 50 creators signed.
- 5+ saves per beta user in the first week.
- 30% weekly return rate.
- Hyderabad seed inventory ready.
